## ðŸ“‹ Abstract
This study presents a comprehensive bibliometric analysis of 1,434 SWMM-related publications indexed in the Web of Science Core Collection between 2000 and 2025. The analysis reveals a marked increase in research productivity since 2010, reflecting a thematic transition from early model development toward solution-oriented approaches emphasizing Low-Impact Development (LID), green infrastructure, and climate resilience.
